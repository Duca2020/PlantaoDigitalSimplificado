FAQ

Esta página é destinada a explicar as principais funcionalidades do sistema de geração de documentos.

Sistema Plantão Digital

O sistema Plantão Digital busca modernizar os serviços do Plantão Funerário, trazendo conforto e praticidade aos cidadãos e eficiência aos plantonistas. A primeira versão é o Plantão Digital Simplificado, adicionando as funções de visualização de tabelas Excel e geração de relatórios para FMS e para SDU-Centro.

Funcionalidades da Geração de Documentos

Essa seção é reservada para apresentar as principais funcionalidades do sistema Plantão Digital Simplificado. São utilizadas imagens ilustrativas das funcionalidades e opções disponíveis, de maneira o usuário possa se guiar durante o processo.

Conhecendo o menu lateral

O Menu Lateral resume as páginas de acesso disponíveis para os plantonistas, em poucos cliques é possível se deslocar entre a página de geração de documentos e a página de ajuda (FAQ).




A tabela integrada

A tabela de dados integrada apresenta uma forma especial de conseguir transferir o conteúdo da tabela do Google Sheets para gerar os relatórios. É importante que os cabeçalhos também sejam copiados, pois essa é a forma como o algoritmo identifica como os dados devem ser separados.




Salvando os dados copiados

Caso queira salvar os dados copiados em uma tabela separada, basta clicar no icone de salvar do lado esquerdo da tabela.




Selecionando Arquivos

O sistema está programado para receber os valores da própria tabela integrada, mas também é possível gerar os relatórios a partir de uma tabela do próprio computador.




Botões de controle

Os botões de controle servem para direcionar o processo, ao clicar em voltar a página é recarregada. O botão de Gerar Relatório FMS irá gerar o relatório no modelo padrão da SDU-Centro com base nos dados copiados na tabela integrada (por padrão) ou a partir de uma tabela carregada (opcional).
